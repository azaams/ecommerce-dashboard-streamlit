Ini adalah proyek akhir (submission) untuk kelas Belajar Analisis Data dengan Python di Dicoding. Proyek ini berfokus pada analisis data E-Commerce Public Dataset, mencakup tahapan data wrangling, exploratory data analysis (EDA), visualisasi data, segmentasi pelanggan (RFM Analysis), dan pembuatan dashboard interaktif.

## Setup Environment - Anaconda
```bash
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
``` 

## Setup Environment - Shell/Terminal
```
mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
